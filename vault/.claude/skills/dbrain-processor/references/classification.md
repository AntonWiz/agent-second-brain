# Entry Classification

> **Домены** — это сферы жизни и работы Антона. Когда прилетает запись — сначала определяем домен, потом куда сохранить и что делать.

---

## Домены → Категории

### Проекты
Задачи, стратегия, воронки, партнёры, результаты по конкретному проекту.

**Keywords:** ГИС, гипноз, Никита, Васту, Акаша, Маркел, Поттериана, Артём, Гудвин, Академия, продюсирование

**→ Category:** task (p1-p2) → Milanote Kanban или `мок/задачи-гтд.md`

---

### YouTube
Идеи видео, сценарии, форматы, трекинг выкладки, подписчики.

**Keywords:** видео, ролик, канал, YouTube, сценарий, Маркел интервью, съёмка, монтаж, Алёна видео

**→ Category:** task (p2) или idea → `проекты/ютуб/`

---

### Контент
Посты, тезисы, Stories — для Telegram-канала «Изумрудный город» и TG Stories.

**Keywords:** пост, Изумрудный город, Stories, тезис, контент, статья, выложить

**→ Category:** idea → `thoughts/ideas/` или task если есть дедлайн

---

### Команда
Делегирование, задачи для Алёны и Вики, новый ассистент.

**Keywords:** Алёна, Вика, делегировать, поручить, ассистент, hh.ru, планёрка, созвон с командой

**→ Category:** task (p2-p3) → `мок/задачи-гтд.md` (контекст: делегировать)

---

### Финансы
Доходы, расходы, вывод, фонды, учёт по проектам.

**Keywords:** деньги, вывод, перевод, фонд, Тинькофф, Альфа, ГИС деньги, Мопс

**→ Category:** task → `финансы/YYYY-MM.md`

---

### Обучение / Рост
Инсайты с разборов, книги, трейдинг, новые навыки.

**Keywords:** Даня, разбор, инсайт, узнал, понял, трейдинг, Алёна индикаторы, книга, курс

**→ Category:** learning → `thoughts/learnings/`

---

### Личное
Семья, здоровье, архетипы, духовные практики.

**Keywords:** Лиза, Алиса, мама, бег, планка, архетип, Гримуар, Высшее Я, рефлексия

**→ Category:** reflection → `thoughts/reflections/` или дневник

---

## Decision Tree

```
Запись содержит...
│
├─ Конкретный проект + действие? ─────────────> TASK (p1-p2)
│  (ГИС, Васту, Поттериана, Гудвин, дедлайн)  → Milanote / GTD
│
├─ YouTube: идея или выложить? ───────────────> TASK или IDEA
│  (видео, ролик, канал, сценарий)             → проекты/ютуб/
│
├─ Делегировать Алёне / Вике? ────────────────> TASK (p2-p3)
│  (поручить, спросить, передать)              → GTD контекст: делегировать
│
├─ Деньги / финансы? ─────────────────────────> TASK
│  (вывод, перевод, учёт)                      → финансы/
│
├─ Инсайт из разговора с человеком? ──────────> LEARNING или PROJECT
│  (узнал, понял, суть от X)                   → thoughts/learnings/
│
├─ Стратегическое / долгосрочно? ─────────────> PROJECT
│  (план, гипотеза, новый продукт)             → thoughts/projects/
│
├─ Личная рефлексия / архетип? ───────────────> REFLECTION
│  (осознал, чувствую, Высшее Я)               → thoughts/reflections/
│
└─ Идея для контента? ────────────────────────> IDEA
   (пост, тезис, Изумрудный город)             → thoughts/ideas/
```

---

## Фильтры решений

Перед сохранением как задачу (особенно стратегическую) — прогони через:

1. **Приближает ли это к 3М/мес?**
2. **Можно делегировать Алёне или Вике — или только я?**
3. **Отрабатывает ли суперсилу?** *(суперсила = извлекать суть из людей и превращать в продукт/систему)*
4. **Это YouTube или через YouTube?** *(тема года — всё через контент)*

Если да на 2+ → повысить приоритет.

## Куда сохранять

| Категория | Место | Приоритет |
|-----------|-------|-----------|
| task (проект) | Milanote Kanban | p1-p2 |
| task (команда/делегат) | `мок/задачи-гтд.md` | p2-p3 |
| task (финансы) | `финансы/YYYY-MM.md` | p1-p2 |
| idea | `thoughts/ideas/` | — |
| reflection | `thoughts/reflections/` | — |
| project | `thoughts/projects/` | — |
| learning | `thoughts/learnings/` | — |

## Именование файлов

```
thoughts/{category}/{YYYY-MM-DD}-короткое-название.md
```

Примеры:
```
thoughts/ideas/2026-02-21-формат-интервью-маркел.md
thoughts/projects/2026-02-21-воронка-гудвин-вебинар.md
thoughts/learnings/2026-02-21-инсайт-разбор-даня-этап3.md
```

## Структура файла мысли

```markdown
---
date: {YYYY-MM-DD}
type: {категория}
domain: {Проекты|YouTube|Контент|Команда|Финансы|Обучение|Личное}
tags: [тег1, тег2]
---

## Контекст
[Откуда пришла мысль — разговор, событие, инсайт]

## Суть
[Ключевая идея одним абзацем]

## Значение
[Что это означает для проекта / дохода / системы]

## Следующее действие
[Конкретный шаг — не абстрактный]
```

---

## Anti-Patterns (ИЗБЕГАТЬ)

- Абстрактные рассуждения без «Следующего действия»
- Задачи типа «подумать о...» — конкретизировать или не сохранять
- Повторы без синтеза — кластеризовать похожие мысли
- Хаотичные списки без приоритетов
